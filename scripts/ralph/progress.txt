(Only relevant if a schema change is needed — this PRD explicitly does NOT change the schema.)

If you change a property id constant, rename the matching `uploads/<uuid>/` directory — the static server at `app.use('/uploads', express.static(...))` (src/app.ts:46) resolves URL path segments directly to filesystem paths.
